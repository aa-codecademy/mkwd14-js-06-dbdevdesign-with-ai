## 📚 Topics

- **Relational model** — tables, keys, relationships, integrity
- **Conceptual & logical design** — ER diagrams, naming, domains
- **Normalization** — functional dependencies, normal forms, trade-offs
- **SQL (DDL & DML)** — types, constraints, queries, migrations mindset
- **Physical design** — indexes, performance-aware modeling
- **Transactions & concurrency** — isolation, locking basics
- **Architecture** — schemas, evolution, documentation
- **AI in the workflow** — drafting schemas, explaining trade-offs, review (with human validation)
